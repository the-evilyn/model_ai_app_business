# Model AI App Business

## Description
Ce projet contient des pipelines de Machine Learning conçus pour analyser et prédire le succès des startups.

## Fonctionnalités Principales
- **Prédiction de Succès :** Modèles de classification binaire pour évaluer la viabilité d'une startup.
- **Analyse de Marché :** Évaluation des données du marché.
- **Analyse de Sentiments :** Extraction de tendances basées sur des textes.
- **Recommandation :** Moteur de recommandation d'experts.

## Installation
1. Clonez ce dépôt.
2. Installez les dépendances requises :
   ```bash
   pip install -r requirements.txt
   ```
